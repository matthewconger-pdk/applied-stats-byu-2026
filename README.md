# Programming With LLMs For Data Practitioners

This repo contains workshop materials for the [50th Summer Institute of Applied Statistics (SIAS)](https://statistics.byu.edu/sias) at Brigham Young University on 17-18 June 2026, taught by [Julia Silge](https://juliasilge.com/).

The workshop covers programming with large language models (LLMs) from R and Python, using [ellmer](https://ellmer.tidyverse.org/) and [chatlas](https://posit-dev.github.io/chatlas/). Topics include system prompt design, token management, tool calling, retrieval-augmented generation (RAG), and Shiny apps.

## Materials

Access all the slides and workshop materials at <https://juliasilge.github.io/applied-stats-byu-2026/>.
